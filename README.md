*a small collection of simple command-line games*

# Features in All Games
- Scorekeeping
- Occasional sarcastic comment

# Rock Paper Scissors
## How to play
- Answer the questions of whether or not you would like to play and how many rounds you would like to play
- Once the game starts, enter the emoji that corresponds to your move
## Features
- The computer randomly picks its move using a random number generator

# Tic Tac Toe
## How to play
- Answer the questions of whether or not you would like to play and how many rounds you would like to play
- Once the game starts, enter the number corresponding to the position your move is
## Features
- The computer semi-randomly picks its move using a random number generator so each round is unique but follows a strategy
- Actual Tic-Tac-Toe Board
